# 🎓 Student AI Usage Dashboard

## 📖 Project Overview

This Power BI dashboard analyses student demographics, learning preferences, and AI tool usage to uncover trends in how students interact with artificial intelligence in their studies.

The dashboard is designed to provide interactive insights through visualisations, KPIs, and filters, allowing users to explore the data by age, degree programme, work mode, and programming language.

---

## 🎯 Objectives

- Analyse student demographics.
- Understand AI usage across different degree programmes.
- Identify trends in programming language preferences.
- Compare AI usage by work mode.
- Build an interactive dashboard using Power BI.

---

## 📊 Dashboard Features
